## Code Block Example

Code block language-aware and with multiple variables

<InlineEditableCodeBlock
  language="yaml"
  template={
  `
    global:
      LogLevel: {{host}}
      LogFileBackups: {{port}}
  `
  }
  variables={{
    host: 'example.teleport.sh',
    port: '443'
  }}
/>

## Editable Command example

Allows to change multiple variables inline and copy

<InlineEditableCommand
  template='curl "https://{{host}}:{{port}}/scripts/install.sh" --cluster {{cluster}}'
  variables={{
    host: 'example.teleport.sh',
    port: '443',
    cluster: 'my-cluster'
  }}
/>
