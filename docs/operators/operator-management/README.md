Once your [SSV Node is up and running](/operators/operator-node/node-setup), you can register your Operator in the network and start earning rewards 🥳

* [**Register Operator**](registration), to start participating in SSV Network.
* [Set Operator Metadata](setting-operator-metadata), to attract more stakers.
* [Withdraw Earnings](withdrawing-earnings), to receive your fees.

#### Other guides
