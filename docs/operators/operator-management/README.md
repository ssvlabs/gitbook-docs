# Operator Management

Once your [SSV Node is running](/operators/operator-node/node-setup), you can register your Operator and start managing it on SSV Network.

#### Essential guides
* [**Register Operator**](registration) - join SSV Network.
* [Set Operator Metadata](setting-operator-metadata) - help Stakers evaluate your Operator.
* [Withdraw Earnings](withdrawing-earnings) - withdraw the fees you have earned.

#### Other guides
* [Updating Operator Fees](updating-operator-fees) - change your fee.
* [Configuring a Permissioned Operator](configuring-a-permissioned-operator) - allow specific validator owners only.
* [Remove an Operator](removing-an-operator) - stop participating with this Operator.
