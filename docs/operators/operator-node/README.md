# Running a Node

The essential guide in this section is [**Setup SSV Node**](/operators/operator-node/node-setup). Other setup options are also available:

- Recreate the same setup manually with the [Manual Node Setup guide](/operators/operator-node/node-setup/manual-setup).
- Use Anchor, an alternative SSV client developed by Sigma Prime. See the [official Anchor documentation](https://anchor-book.sigmaprime.io/running_node.html).
- Use [eth-docker](https://eth-docker.net/Support/SSV/), [Stereum Launcher](https://stereum.net/), or [Dappnode](https://docs.dappnode.io/docs/user/staking/ethereum/dvt-technologies/ssv-network/).

The rest of this section covers optional yet useful guides:

### [**Setup SSV Node**](/operators/operator-node/node-setup)

Everything you need for a successful setup and reliable performance.

* [Hardware Requirements](/operators/operator-node/node-setup/hardware-requirements) - check that your hardware is sufficient.
* [Best Practices](/operators/operator-node/node-setup/best-practices) - improve performance and reliability.
* [Node Configuration Reference](/operators/operator-node/node-setup/node-configuration-reference) - advanced configuration reference.
* [Manual Node Setup](/operators/operator-node/node-setup/manual-setup) - manual installation guide.

### [**Setup Sidecars**](/operators/operator-node/setup-sidecars)

Optional sidecars that can expand functionality or improve your Operator profile.

* [Enable DKG](/operators/operator-node/setup-sidecars/enabling-dkg/) - participate in DKG ceremonies and attract more validators.
* [Configure MEV](/operators/operator-node/setup-sidecars/configuring-mev) - increase rewards for block proposals.
* [Configure Primev](/operators/operator-node/setup-sidecars/configuring-primev) - an alternative MEV-related flow that includes preconfirmations.
* [Configure commit-boost](/operators/operator-node/setup-sidecars/configuring-commit-boost) - an alternative MEV-related flow that can support additional services.
* [Configure SSV Remote Signer](/operators/operator-node/setup-sidecars/remote-signer) - separate key management from the SSV Node.

### [**Monitoring**](/operators/operator-node/monitoring)

Monitoring helps you detect problems early and understand node performance.

* [Dashboard Runbook](/operators/operator-node/monitoring/dashboard-runbook) - install and use the Grafana dashboard.
* [Metrics Index](/operators/operator-node/monitoring/metrics-index) - reference for SSV metrics.

### [**Maintenance**](/operators/operator-node/maintenance)

Guides for diagnosing issues and maintaining your node.

* [Troubleshooting](/operators/operator-node/maintenance/troubleshooting) - identify and fix common issues.
* [Common Errors](/operators/operator-node/maintenance/common-errors) - look up common log messages and likely causes.
* [Node Migration](/operators/operator-node/maintenance/node-migration) - move an SSV Node to a new machine.
* [DKG Node Migration](/operators/operator-node/maintenance/dkg-operator-migration) - move a DKG Node to a new machine.
