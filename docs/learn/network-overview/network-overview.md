# Network Overview

SSV Network has three main actor groups: stakers, operators, and DAO participants. Separately, SSV stakers support the oracle system; for more on that role, see the [SSV Staking section](/ssv-staking).

### Stakers

Services or individual ETH holders that use SSV/DVT technology for improved liveness, security, and decentralization of their validators.

Stakers pay a fee in ETH to operators for managing their validator(s).
- [Validator Concepts](/learn/network-overview/validators)
- [Cluster Concepts](/learn/network-overview/clusters)
- [Stakers Quickstart page](/stakers)

### Operators

Operators provide hardware infrastructure, run the SSV protocol, and help maintain validators and the overall health of SSV Network.

Operators determine their fees for services and charge stakers for managing their validator(s).
- [Operator Concepts](/learn/network-overview/operators)
- [Operator Performance](/learn/network-overview/performance)
- [Cluster Concepts](/learn/network-overview/clusters)
- [Operators Quickstart page](/operators)

### DAO

The SSV Network DAO decentralizes ownership and governance of the protocol and treasury, with SSV as the native token. It exercises this mandate through governance functions and by executing Snapshot proposals passed by token holders.

For more information, see [Governance](/learn/network-overview/governance).
