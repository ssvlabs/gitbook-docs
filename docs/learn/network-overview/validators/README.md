# Validator Concepts
Distributed validators are managed within clusters, the groups of operators selected to operate them.

To run a validator through SSV Network, a staker must distribute the validator key to the selected cluster and register it with the network’s smart contract. This can be done through the [Web App](https://app.ssv.network/), through the smart contracts, or with [developer tools](/developers).

This section covers validator-related concepts. For action guides, use the staker guides linked throughout the docs.

**We strongly recommend checking [Cluster Concepts](/learn/network-overview/clusters)**, since every validator registered on SSV Network is managed by a cluster.
