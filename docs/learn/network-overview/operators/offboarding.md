# Operator Offboarding

An operator can choose to leave the network at any time. Choosing to remove an operator will:

- Cease operation of all their managed validators, resulting in reduced fault tolerance for their managed validators and could put them at risk.
- Remove themselves from the network, after which they will no longer be discoverable by other validators.
- Immediately stop receiving future earnings from their managed validators' operation.
- All ETH + SSV earnings are automatically settled and transferred on removal.

:::warning
Please be aware: this process is **irreversible**, and removed operators cannot reactivate or register the same operator again in the future.

Attempting to register an operator using the same private key will result in an error. In that case, operator registration must be done with a new key.
:::
