# Operator Concepts

To join the network as an operator a user must run an SSV node.

This is software containing the SSV protocol implementation and integration to the network's smart contracts.

This can be done by installing the [node software](/operators/operator-node/) on a machine followed by registering the operator either through the [webapp interface](https://app.ssv.network/) or the [smart contract](/developers/smart-contracts/ssvnetwork#operator-methods).
