# Operator Concepts

To join the network as an operator, an operator owner must run an SSV Node.

This software contains the SSV protocol implementation and integration with the network’s smart contracts.

This subsection explains the main concepts related to operators on SSV Network. To become an operator, install the [node software](/operators/operator-node/) and then register through the [Web App](https://app.ssv.network/) or the [smart contract](/developers/smart-contracts/ssvnetwork#operator-methods).
