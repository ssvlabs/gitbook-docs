# Validator Offboarding

If you need to offboard validators, follow these steps:

1. [Exit validators](exiting-a-validator.md).
2. Wait until validators are fully exited.
3. [Remove validators](removing-a-validator.md) from your cluster. 
4. Once they are exited and removed, [withdraw your ETH balance and refundable liquidation collateral](withdrawing-eth.md).

:::note Automations
All of these steps can be automated programmatically. If you wish to do so, please check out [Developers Tutorials page](/developers/examples/).
:::
