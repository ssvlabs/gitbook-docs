# Validator Offboarding

:::note Automations
These steps can also be automated. See the [Developers Tutorials](/developers/examples).
:::

Follow this flow in order:

1. [Exit the validator](exiting-a-validator).
2. Wait until the validator is fully exited on Ethereum.
3. [Remove the validator](removing-a-validator) from the cluster.
4. [Withdraw ETH and refundable collateral](withdrawing-eth), if applicable.
