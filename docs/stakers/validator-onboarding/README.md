# Validator Onboarding

![Key Splitting Flow](/img/flow_chart_keysplit.png)

Tutorials here will help to migrate your validators to SSV network from a vanilla setup.

Follow our guides in this order:
1. [Split Keys](split-keys.md)
2. [Calculate Costs](calculate-costs.md)
3. [Register Validators](register-validators.md)
4. [Post-Onboarding Checks](post-onboarding.md)

:::note Automations
Most of these steps can be automated programmatically. If you wish to do so, please check out [Developers Tutorials page](/developers/examples/).
:::
