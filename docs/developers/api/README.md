# API

Two ways to fetch SSV Network's on-chain data is by using [**SSV Subgraph**](ssv-subgraph) or [**SSV RESTful API**](ssv-api):

- [**SSV Subgraph**](ssv-subgraph) is recommended to be used in automations/scripts, as it is more reliable and highly customizable. Check out [Subgraph Examples](subgraph-examples) to find common subgraph reuqests.
- [**SSV API**](ssv-api) is recommended for quick tests or manual requests, as its request types are limited to existing ones and the API is not built to withold a lot of requests.
