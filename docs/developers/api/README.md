# API

Use one of these two options to retrieve SSV Network data:

- [**SSV Subgraph**](ssv-subgraph) is the recommended option for scripts, automations, and custom queries. See [Subgraph Examples](subgraph-examples) for common requests.
- [**SSV API**](ssv-api) is better for quick checks and simple manual requests. It offers a fixed set of endpoints and is not intended for heavy query volume.
